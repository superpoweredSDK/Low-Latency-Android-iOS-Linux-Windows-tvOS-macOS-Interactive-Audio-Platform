Detailed Description

Instead of circular buffers and too many memmove/memcpy, this object maintains an audio buffer "chain". You can append, insert, truncate, slice, extend, etc. this without the expensive memory operations. Check the SuperpoweredOfflineProcessingExample project on how to use.

Parameters
sampleLengthThe number of samples inside this list.
