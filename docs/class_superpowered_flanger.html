Detailed Description

One instance allocates around 80 kb memory.

Parameters
wet0.0f to 1.0f. Read only.
depthMsDepth in milliseconds, 0.3f to 8.0f (0.3 ms to 8 ms). Read only.
depth0.0f to 1.0f (0.0 is 0.3 ms, 1.0 is 8 ms). Read only.
lfoBeatsThe length in beats between the "lowest" and the "highest" jet sound, >= 0.25f and <= 64.0f. Read only.
bpmSet this right for a nice sounding lfo. Limited to >= 60.0f and <= 240.0f. Read-write.
clipperThresholdDbThe flanger has a SuperpoweredClipper inside to prevent overdrive. This is it's thresholdDb parameter.
clipperMaximumDbThe flanger has a SuperpoweredClipper inside to prevent overdrive. This is it's maximumDb parameter.
stereoStereo/mono switch. Read-write.
