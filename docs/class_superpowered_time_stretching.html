Detailed Description

One instance allocates around 100 kb. Check the SuperpoweredOfflineProcessingExample project on how to use.

Parameters
rate1.0f means no time stretching. Read only.
pitchShiftShould be -12 (one octave down) to 12 (one octave up). 0 means no pitch shift. Read only.
numberOfInputSamplesNeededHow many samples required to some output. Read only.
