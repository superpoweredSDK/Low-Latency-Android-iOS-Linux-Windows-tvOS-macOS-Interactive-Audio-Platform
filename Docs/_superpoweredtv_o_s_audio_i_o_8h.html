typedef bool(* audioProcessingCallback_C) (void *clientdata, float **buffers, unsigned int outputChannels, unsigned int numberOfFrames, unsigned int samplerate, uint64_t hostTime)

You can have an audio processing callback in Objective-C or pure C. This is the pure C prototype.

Returns
Return false when you did no audio processing (silence).
Parameters
clientDataA custom pointer your callback receives.
buffersInput-output buffers.
inputChannelsThe number of input channels.
outputChannelsThe number of output channels.
numberOfFramesThe number of frames requested.
samplerateThe current sample rate in Hz.
hostTimeA mach timestamp, indicates when this buffer of audio will be passed to the audio output.
